# Runtime Transforms

Runtime-facing authoring helpers no longer live in `lil`.

- Core IR functionality stays in `github.com/neutrome-labs/lil`.
- Go authoring helpers such as `manip`, `slwin`, and `kvtools` now live in
  `github.com/neutrome-labs/lilsdk-go`.

See [../README.md](../README.md) and [../../sdk/lilsdk-go/README.md](../../sdk/lilsdk-go/README.md)
for current import paths and examples.
